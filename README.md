<h1 align="center">Lista De Produtos</h1>

### INSTALAÇÃO
1. Clone o repositório

   ```sh
   git clone https://github.com/NiloMarcos/ListOfProducts.git
   ```

2. Entre na pasta

   ```sh
   cd ListOfProducts
   ```

3. Instale as dependências

   ```sh
   yarn
   ```

   ou

   ```sh
   npm i

### Iniciar a aplicação
1. Abra o terminal e rode a aplicação com o seguinte comando
   ```sh
   yarn dev
   ```
